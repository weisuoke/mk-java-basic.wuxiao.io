# Maven 介绍

- Maven 是项目管理工具，对软件项目提供构建与依赖管理
- Maven 是Apache下的Java开源项目
- Maven为Java项目提供了统一的管理方式，已成为业界标准

## Maven 核心特性

- 项目设置遵循统一的规则，保证不同开发环境的兼容性
- 强大的依赖管理，项目依赖组件自动下载、自动更新

- 可扩展的插件机制，使用简单，功能丰富
