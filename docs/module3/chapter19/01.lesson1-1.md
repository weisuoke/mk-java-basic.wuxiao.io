# 框架的作用

## 软件开发中的框架

- 框架是可被应用开发者定制的应用骨架
- 框架是一种规则，保证开发者遵循相同的方式开发程序
- 框架提倡”不要重复造轮子“，对基础功能进行封装

## 框架的优点

- 极大提高了开发效率
- 统一的编码规则，利于团队管理
- 灵活配置的应用，拥有更好的维护性
