# SqlSessionFactory

- SqlSessionFactory是MyBatis的核心对象
- 用于初始化MyBatis，创建SqlSession对象
- 保证SqlSessionFactory在应用中全局唯一

## SqlSession

- SqlSession是MyBatis操作数据库的核心对象
- SqlSession使用JDBC方式与数据库交互
- SqlSession对象提供了数据表CRUD对应方法
