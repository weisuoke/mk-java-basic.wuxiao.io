# 面向对象

1. 什么是对象
2. 什么是面向对象
3. 什么是类
4. 类和对象的关系

 ## 什么是面向对象

- 人关注对象
- 人关注事务信息

## 类和对象

- 类是模子，确定对象将会拥有的特征（属性）和行为（方法）
- 对象是类的实例表现
- 类是对象的类型
- 对象是特定类型的数据

## 属性和方法

- 属性：对象具有的各种静态特征
- “对象有什么”
- 方法：对象具有的各种动态行为
- “对象能做什么”

## 类和对象的关系

- 类
  - 抽象的概念
  - 模板
- 对象
  - 一个看得到、摸得着的具体实体
